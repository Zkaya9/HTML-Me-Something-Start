Projects:

  • Mars Rover Project - This project simulates commanding the Mars rover Curiosity by implementing three classes: Command, Message, and Rover. The Command class defines instructions for the rover, the Message class bundles commands for delivery, and the Rover class handles received commands to update its properties. The project follows Test-Driven Development (TDD) principles, where unit tests are written before coding each feature.
  • Scrabble Scorer Project - This project enhances a Scrabble scoring program by adding multiple scoring algorithms and allowing users to choose which one to use interactively. It includes functions for prompting the user to input a word, calculating the score using different scoring algorithms, and transforming the scoring system. The project is organized into tasks, starting with writing an initial prompt function and ending with submitting the completed work.
  • Candidate Testing Project - A console application for conducting short quizzes with candidates. The project prompts users to input their names, answers five questions, and calculates their overall score. Passing requires scoring >= 80%. The project is divided into three parts: creating the quiz, expanding it, and grading the results.
